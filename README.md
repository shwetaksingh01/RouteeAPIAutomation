
# RestAssuredFrameworkForRouteeAPIs

## Build the Project
```
mvn clean install
```

## Run the Test
```
mvn clean test
```

## Projects/Modules
This project contains a number of modules, but here are the main ones you should focus on and run: 
```
`src/main`: For base utilities 
```
```
`src/test`: Test apis
```
```
`src/main/resources`: env.properties file
```
